What is SDAG technology
SDAG(Super directed acyclic graph) based on existing DAG technology to construct a hierarchical chain entanglement network. Each level of blockchain network stores the corresponding data, similar to the country -> Province -> City -> County -> District, each administrative center to manage the data, between provinces and between countries and countries, each other Care about each other's data. Separation of the entire network of block data, each block network only care about their own needs of the block data, you can reduce a large amount of data redundancy.
Why support both Transaction proof of transaction and POS proof of ownership?
Support for free transactions and fees transactions
Free trade, record common data, do not need transaction costs, and IoT devices produce massive data scenarios fit. Charged transactions apply to high value data transactions in the Internet of Things. Record data in the high-level block network, better liquidity and higher security.
Has unlimited branch network capacity
An algorithmic mechanism monitors the economic value of the block network. When the economic value of the network reaches a certain threshold, it will automatically trigger the next new block verification needs charging. At this moment, if the user chooses to continue to charge the network transaction in this level block, if the user wants to continue to use the free service, the network system will branch a new sub-block network from the current block network and write the new transaction Sub-block network, reaching unlimited branch block network capabilities.
Password economics balance
The combination of free transactions and paid transactions can form a relationship of mutual checks and balances. Form a hierarchy of data distribution by charging and freeing. Free transaction data is only stored in the lower level or the end level block network, the charge transaction is stored in the high level block network, the higher the transaction fee, the higher the storage network level. This avoids junk data blocking high-level block networks and also reduces the amount of data in high-level block networks.
Advantage
Reducing the cost of the Internet of things
Block chain technology provides a point to point direct interconnection way for data transfer. The whole Internet of things solution does not need to introduce large data centers for data synchronization and management control, thereby reducing the operation cost of the Internet of things.
Big Data Management
As far as large data analysis and mining are concerned, the application of blockchain expands the scale and richness of the data. The technology of block chain makes the quality of data get unprecedented trust. It ensures the correctness of data analysis and data mining, and realizes the prediction, analysis and action of big data. It will become the cornerstone of the digital economic era.
Security and Transparency
Blockchain relies on discrete decentralized computer clusters to run, and its fair consensus mechanism to ensure the security of the network, you can view the history of transactions, to ensure the transparency of information.
